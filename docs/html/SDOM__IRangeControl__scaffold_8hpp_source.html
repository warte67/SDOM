150 //
151 // Display objects that cache renderer-owned resources (e.g., SDL_Texture*)
152 // must invalidate and rebuild those resources whenever the SDL device
153 // is recreated or when logical size/format changes. Core guarantees to
154 // broadcast onWindowResize(logicalW, logicalH) in two cases:
155 // 1) In response to SDL window resize events
156 // 2) Immediately after Core reconfigures SDL (e.g., when setters like
157 // setPixelFormat/setWindowFlags trigger a device rebuild)
158 //
159 // To remain robust across device rebuilds, overrides should:
160 // - Prefer releasing GPU resources in onUnload() and (optionally)
161 // pre-warm in onLoad().
162 // - For onWindowResize, drop-only any cached SDL_Texture pointer
