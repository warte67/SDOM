Detailed Description

Abstract interface for displayable objects in SDOM.

Provides the core API for all objects that can be rendered, positioned, and interacted with in the SDOM system. Implementations must define methods for drawing, updating, event handling, and layout management.

Typical responsibilities:

  • Rendering to a display surface
  • Managing position, size, and bounds
  • Handling user input and events
  • Supporting layout and anchoring
Note
This is a pure interface; all methods are expected to be overridden by derived classes.
