10Set the dirty flag when:
11- the Label bounds change (width, height, position).
12- the text changes.
13- the FontStyle changes.
14- the parentâ€™s bounds change (if the Label is anchored or its layout depends on the parent).
15- the font resource changes (switching fonts or font size).
16- wordwrap or auto_resize flags change.
17- padding or margin changes.
18- visibility toggles (hidden/shown).
19- DPI or scaling factor changes (if supported).
20- the Label is moved between stages or layers (if rendering context changes).
21
22When to rebuild the cachedTexture_:
23- the Cores::getPixelFormat() changes (e.g. due to display mode change).
24- the Labels bounds change (width, height).
25
26When the dirty flag is set:
271) recreate the Labels texture (if needed)
282) render to the cachedTexture_
293) clear the dirty flag
