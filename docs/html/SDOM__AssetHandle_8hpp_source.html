5// @brief Lightweight, non-owning handle for accessing assets within SDOM.

7// üß≠ Purpose:
8// Provides safe, lightweight access to assets (e.g., textures, sounds,
9// sprite sheets) managed by SDOM::Factory and SDOM::Core. The handle is
10// non-owning and idempotent, suitable for exposure to Lua as `AssetHandle`.
