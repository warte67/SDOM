Detailed Description

Core interface for all data-driven objects in SDOM.

Provides property and command registration, dynamic serialization/deserialization, and runtime manipulation of object state via JSON. Enables introspection, scripting, and editor integration for resource and display object management. Forms the foundation for robust data binding, configuration, and automation.

Example usage:

// Register a property
registerProperty("x",
[](const IDataObject& obj) { return obj.x_; },
[](IDataObject& obj, const Json& val) { obj.x_ = val.get<int>(); });
// Register a command
registerCommand("reset", [](IDataObject& o, const Json&) { /-- reset logic --/ });
