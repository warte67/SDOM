◆ SDOM_EventPhase

enum SDOM_EventPhase
Enumerator
SDOM_EventPhase_Capture 

Event is traveling from the root down toward the target (capture phase).

SDOM_EventPhase_Target 

Event has reached the target object; handlers on the target run here.

SDOM_EventPhase_Bubbling 

Event is traveling upward from the target back toward the root (bubble phase).
